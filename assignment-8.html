Intro to Computational Media Portfolio

Objective

Create a multi-page website from scratch using only HTML & CSS.

Description

For the final project, you will create a portfolio website featuring all the assignments you have created in Intro to Computational Media. Every webpage should feature screenshots, link, and descriptions that provide contexts to the work you've done in this semester.
